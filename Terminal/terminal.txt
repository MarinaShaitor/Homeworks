1) Посмотреть где я - pwd
2) Создать папку - mkdir home
3) Зайти в папку - cd home
4) Создать 3 папки - mkdir f1 f1 f3 
5) Зайти в любую папку - cd f2 
6) Создать 5 файлов (3 txt, 2 json) - touch file1.txt file2.txt file3.txt file4.json file5.json
7) Создать 3 папки - mkdir mar1 ri2 na3
8. Вывести список содержимого папки - ls -la
9) + Открыть любой txt файл - vim file1.txt
10) + написать туда что-нибудь, любой текст. - i (появляется insert)
11) + сохранить и выйти. - нажать кнопку esc : wq
12) Выйти из папки на уровень выше - cd ..
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку. - mv f2/{file1.txt,file2.txt} f1 
14) скопировать любые 2 файла, которые вы создали, в любую другую папку. - cp f2/{file1.txt,file2.txt} f1 
15) Найти файл по имени - find . -name file1.txt
16) просмотреть содержимое в реальном времени. - tail -f file1.txt 
17) вывести несколько первых строк из текстового файла - head -3 file3.txt
18) вывести несколько последних строк из текстового файла - tail -5 file1.txt 
19) просмотреть содержимое длинного файла (команда less) изучите как она работает. - less file4.json
20) вывести дату и время - date
